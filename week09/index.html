Notes:

  1. Javascript: Novice to Ninja - Chapter 09: The Window Object
    1. Global variables are actually properties of the window obj. i.e. x=6 is the same as window.x=6
    2. Pay close attention to window.location there are a number of useful properties there. host, hostname, pathname, etc.
    3. Several important ones: search, hash, hostname
    4. window.history.go(1); // goes forward 1 page
window.history.go(0); // reloads the current page
window.history.go(-1); // goes back 1 page
    5. Don't use pop-ups. Even though the methods exist in javascript they are often disabled, or the user is given an option to disable them. Find another more elegant way of doing the task. AJAX for example.
    6. DOM exists within the window.
    7. Animation is possible with Javascript but really CSS can do it to and it would probably be a better place for it.
  2. Javascript: Novice to Ninja - Chapter 14: HTML5 APIs
    1. Using data- attributes: 
      <div id='hero' data-powers='flight superSpeed'>
    Superman
</div>
      const superman = document.getElementById('hero');
const powers = superman.dataset.powers;
<< 'flight superSpeed'
    2. Video and Audio tags have the same general controls.
    3. Place text inbetween the audio/video tags to leave a message if the tag isn't supported.
    4. Look into the canvas element as a wet sig possibility.
