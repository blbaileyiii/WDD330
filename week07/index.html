  1. Javascript: Novice to Ninja - Chapter 11: Further Functions
    1. function sayHello(); calls an unspecified obj "this". You can control what "this" refers to by sayHello.call(someotherobj) additional parms are comma'ed after the 'someotherobj' and passed to the function.
    2. .apply() works the same way but additional parms must be in an array.
    3. .cache[x] attached to and within a function call results in placing the value in a cache array attached to the function. This way you can retrieve past values. It is a key value pair assignment.
    4. A function can redefine itself. sayHello(){ sayHello = function (){ something else...;}}
    5. Redefining a function idea for me... It can be used to run a specific way the first time... and then change for the second iteration...
    6. WARNING: redefining a function drops prior assignments to properties... it starts "clean"...
